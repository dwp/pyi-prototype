Bank account details: roll number

Some accounts need a roll or reference number for payments

You’ll find this on your card, statement or passbook.

What is this?
Most standard bank accounts have an account number and a sort code. But some building society accounts also have a roll number - a reference code with letters and numbers. If you have one, you’ll have used it to make payments.
